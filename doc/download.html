Version

Current version is 1.3.0, released the 17th February 2008.

1.3.0 introduces automatic layout management for the filter header. On previous versions, the user had to place manually the header in the Gui container, as detailed here, while now is is enough to instantiate the header and set the associated table: 

TableFilterHeader filterHeader = new TableFilterHeader(table);

Version 1.3.0 keeps backwards compatibility: if the TableFilterHeader is explicitely included in some Gui container, the default behaviour to automatically setup the filter header close by the table header is deactivated.
